LazyCure Overview

LazyCure is a program for personal time logging, spent time analysis, working time usage efficiency calculation. It is the best tool for time logging of working activities on computer, but could also be used for the-whole-day time logging or for specific activities time tracking.

How to use

  1. When you finish doing any activity, specify the activity on the main LazyCure form and press 'Done'.
  2. Continue doing this everytime you switching between different activities.

Simple usage scenario

Let's consider the simple scenario of LazyCure usage. You come to your office, start computer and LazyCure. You prepare working place, start Outlook and Total Commander (or other programs which you are using every day). Then you open LazyCure, enter "prepare to work" in text field and press "Done". Then you process your email inbox for 20 minutes, press Ctrl+F12 (it is a hot key for activation LazyCure), enter "process inbox" and press "Done". In the same manner you continue to log your activities during the day, specifying them in LazyCure everytime you are done with one activity and ready to perform another.

LazyCure will store information about your activities in Time Log Editor - it is available via menu Show->Time Log. At the end of a day it will be interesting to see summary about activities - how much time you spent on each activity during the day. For example, if you have a talk with Andrei Pretkel thrice a day with duration 0:18, 1:06 and 0:38, Activities Summary, available at Show->Summary will tell you that you spent totally 2 hours and 2 minutes on this activity.

It could happen that you forget to log switching activity, in such a case you can open Time Log Editor and manually add passed activities, specifying their start, duration and end time. It is necessary to specify only two values: start time and duration, start time and end time, duration and end time - the third time value will be calculated automatically. Also you can correct existing info in Time Log Editor in order to make it more correct.

Recommendations

  • Press 'Done!' button each time you switch between activities, even if it is planned to spent seconds on the next activity. It will help you to discover your time thieves.

Complete list of features

Here is a complete list of LazyCure features:

  • ability to perform time logging;
    • start, end time and duration is calculated automatically;
    • possibility to edit time logs;
    • possibility to enter time logs manually;
    • ability to open time logs for past days;
  • features for convenient and fast time logging:
    • lives in tray
    • activated by hot key (Ctrl+F12) or click on tray icon;
    • switch activity by hot key (Ctrl+Alt+Shift+F12) or press ENTER in main window;
    • hide main window on press ‘ESC’ or ‘X’;
    • ability to select activity from list of latest activities;
    • auto completion for activities;
    • show current time with seconds;
    • show current activity start and duration time;
    • possibility to specify finished activity from tray;
    • auto generation of unique activity name if it has not been specified;
    • posibility to switch activity during computer locking;
  • provide reports and metrics for spent time analysis:
    • activities summary – summary time spent on each activity;
    • tasks summary – group similar activities in tasks and see summary time spent on each task;
    • show working activities time;
    • show time for selected activities or tasks;
    • working time usage efficiency – time spent on working activities / time spent on working place;
  • additional features:
    • works under Linux;
    • notification by flashing icon in tray after long time of inactivity;
    • export timelogs to xml;
    • store links between activities and tasks;
    • autosave;
    • options for creation your own LazyCure configuration;
    • possibility to redefine hot keys; 
    • tasks editor provide possibility to add/edit/delete tasks;
    • notification about not logged activity when computer is shutting down;
    • switch time logs at midnight;
    • restore main window position after restart;
    • info about current activity is shown when cursor is placed on tray icon;
    • possibility to post activity to Twitter.
